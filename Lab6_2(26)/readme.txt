Программа хранит данные в бинарном файле, который, при отсутствии, будет создан.
Работа осуществляется в 3х режимах  - просмотра, изменения и добаления записей. Предлагается табличный вид и расширенный - одна запись на страницу, каждое поле отображается в отдельной строке, и вид для составления рапортов. Можно формировать набор инфомации, которая помещается в рапорт, допечатывать вручную любые данные, заголовки, сохранить в RTF файл. Переключение между ними - как между вкладками. Окно для изменения фильтров выбора(поле (поля)  + точная дата, либо с какой-то даты до сегодняшней).
